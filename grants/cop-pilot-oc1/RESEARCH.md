## 8. TECHNICAL REQUIREMENTS (Platform Integration)

### Two Primary Workflows for Open Call Participants:

**Workflow 1 - Onboarding New Services** (for Service Providers):
- Publish application to git-based repository (container images + deployment manifests: Helm charts, K8s manifests, Docker Compose)
- CI pipeline auto-translates to standardized service blueprints
- Service ordered via LLM-based portal chat
- Application must be containerized and expose performance metrics (latency, throughput, KPIs)

**Workflow 2 - Integrating New Infrastructure Domains** (for Infrastructure Owners):
- Register private domain (testbed, lab, compute cluster) via portal
- Auto-Pilot process establishes zero-trust encrypted connections via SIF
- Must run COP-PILOT Domain Orchestrator (DO) and Data Management (DM) components
- Use compatible infrastructure controllers (e.g., Kubernetes)

### Key Platform Technologies:
- **Business Portal**: Next.js, React, Keycloak, LangChain, Ollama/vLLM
- **E2E Service Orchestrator (ESO)**: Next.js, React, Apache Quarkus, Prometheus, Grafana, Apache Kafka
- **Domain Orchestrator (DO)**: Angular, Spring Boot, Keycloak, Flowable, Apache ActiveMQ, MCP
- **Data Management**: Orion/Scorpio/Stellio Context Broker, Eclipse Arrowhead
- **Secure Integration Fabric (SIF)**: OpenZiti (Go), X.509/PKI, mTLS
- **Compute Controller**: Kubernetes (vanilla, microk8s, k3s, k9s), OpenShift, OCM, ColonyOS

### Standards Compliance Required:
- TMForum (TMF) APIs for service/resource orchestration
- ETSI ZSM for zero-touch network management
- NGSI-LD / NGSI-v2 for data management
- FIWARE, Eclipse Arrowhead for IoT
- CNCF SPIFFE, OAuth 2.0, OpenID Connect for security

## 14. SME CHECKLIST - What to Prepare

The SME Checklist requires detailed financial and ownership data:

**Basic Data** (from last approved accounting period):
- Number of employees (annual working units)
- Annual turnover (EUR)
- Annual balance sheet total (EUR)
- Duration and closing date of last accounting period

**SME Thresholds** (must be under ALL):
- < 250 employees
- < EUR 50M annual turnover
- < EUR 43M annual balance sheet

**Ownership Structure** (must declare):
- **Linked enterprises** (upstream/downstream): majority voting rights, dominant influence, control
- **Partner enterprises** (upstream/downstream): 25%+ capital/voting rights holders
- Special categories exempt from calculation: public investment corps, VCs, business angels (<EUR 1.25M), universities, non-profit research centres, institutional investors, small local authorities

**If linked/partner enterprises exist**: Must provide their employee count, turnover, and balance sheet data to calculate consolidated SME status.

**Who needs to prepare this**: Finance department / CFO — requires access to latest approved accounting data and full ownership structure information.
